# Бинарная и многоклассовая классификация: сравнение моделей

Демонстрация машинного обучения на двух задачах:
1. **Бинарная классификация**: апельсины vs грейпфруты (`oranges-vs-grapefruit` из Kaggle)
2. **Многоклассовая классификация**: вина (`sklearn.datasets.load_wine`)

Сравниваются модели:
- Логистическая регрессия
- kNN
- Дерево решений
- SVM
- Random Forest

Оцениваются метрики: Accuracy, Precision, Recall, F1, ROC AUC, PR AUC, Specificity.
